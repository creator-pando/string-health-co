# 강건한건강

> 건강하게 일하고 강하게 성장하는 플랫폼

강건한건강은 전문 프리랜서와 기업을 연결하는 서비스 외주 플랫폼입니다.
필요한 서비스를 빠르고 신뢰있게 연결합니다.

## 서비스

- 🤝 **전문가 매칭** — 검증된 전문 프리랜서와 프로젝트를 빠르게 연결합니다.
- 💼 **외주 서비스** — 개발, 디자인, 마케팅 등 다양한 분야의 외주를 지원합니다.
- 🛡️ **안전한 거래** — 투명한 계약과 안전한 정산으로 신뢰 있는 거래를 보장합니다.
